# タイポグラフィ

テキスト表示に関するルールをまとめます。

## フォントファミリー

可読性や視認性を高めるためにサンセリフ体とします。<br />
書体（フォント）には、インターネット環境があれば、どなたでも無償で利用できる「オー プンソース書体」を選択。<br />
また数あるオープンソース書体のなかから、環境によって表示できない文字をなくす挑戦から生まれた「Noto Sans」を採用しました。<br />
Google Font で提供されているウェブフォント、Noto Sans Japanese を使用してください。

サンプル：

<div>
  <Typography variant="body">0123456789abcdABCDあいうえおアイウエオ</Typography>
  <Typography variant="body" sx={{ fontWeight: "bold" }}>
    0123456789abcdABCDあいうえおアイウエオ
  </Typography>
</div>
